# NextTV Viewer - Quick Start

## Pre-loaded Playlists

The app comes with **two playlists** automatically loaded on first launch:

### 1. Sample Channels (30+ channels)
Demo/test channels organized by category:
- UK, News, Sports, Entertainment, Documentary, Kids, Music

### 2. Link4TV Channels (Real IPTV)
Full IPTV playlist from: `http://link4tv.me:80/playlist/COCHJNAR01/VYa7uMUeFT/m3u_plus?output=hls`
- Hundreds of live TV channels
- Multiple countries and languages
- Sports, movies, news, entertainment

## Getting Started

1. **Launch app** → Both playlists load automatically
2. **Select playlist** → Link4TV is set as default
3. **Browse channels** → Scroll through hundreds of channels
4. **Play channel** → Tap any channel to watch
5. **Previous channel** → Play 2 channels, press "Previous" to return
6. **Switch playlists** → Go to Settings → Select "Sample Channels" or "Link4TV Channels"

## Features

- ✅ **Previous Channel Button** - One-tap return to last channel
- ✅ **Channel Groups** - Organized by category
- ✅ **Favorites** - Star your favorite channels
- ✅ **Search** - Find channels by name
- ✅ **EPG Support** - Electronic Program Guide (XMLTV)
- ✅ **Multiple Playlists** - Switch between playlists in Settings
- ✅ **Remote Control** - Full TV remote support

## Installation

```bash
npm install
npm run android
```

## Adding More Playlists

1. Enter M3U URL in home screen input
2. Tap "Add" button
3. New playlist loads and appears in Settings

That's it! Start watching immediately with pre-loaded channels.
