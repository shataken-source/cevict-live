1. Architectural Strategy: The "Static-First" Hybrid
To ensure maximum speed for users in high-stress situations (like just having lost a pet), we will utilize a hybrid approach:

Static Generation (SSG): All informational and high-traffic entry pages (About, Help, Success Stories) will be pre-rendered as static HTML for instant loading.

Dynamic Client Layers: The "Report Lost Pet" and "AI Image Match" features will remain dynamic client components to handle real-time data and file uploads.
