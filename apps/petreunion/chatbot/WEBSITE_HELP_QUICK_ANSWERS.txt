PetReunion Website Help - Quick Answer Library

Common Questions - Instant Answers

"How do I report a lost pet?"
"Go to petreunion.com/report/lost. Fill out the form with:
1. Pet Type (required: Dog or Cat)
2. Color (required)
3. Date Lost (required)
4. Location (required: city AND state, like 'Columbus, Indiana')
5. Contact info (optional but recommended)
Click 'Submit Report' when done!"

"I'm getting a location error"
"Location needs BOTH city AND state. Examples:
✅ 'Columbus, Indiana'
✅ 'Columbus IN'
✅ 'Indianapolis, IN'
❌ 'Columbus' (missing state)
❌ 'Indiana' (missing city)"

"Date error - can't select date"
"Make sure:
- Date is not in the future
- Date is not more than 10 years ago
- You're clicking the date field
- Try typing date manually: YYYY-MM-DD format"

"Email format error"
"Email must have: name@domain.com
✅ yourname@email.com
✅ user.name@example.org
❌ yourname@email (missing .com)
❌ @email.com (missing name)"

"Phone number error"
"Phone needs 10-15 digits. Formats that work:
✅ (555) 123-4567
✅ 555-123-4567
✅ 5551234567
Spaces and dashes are fine - they're removed automatically."

"Form won't submit"
"Check for red error messages. Common issues:
1. Missing required field (Pet Type, Color, Date, Location)
2. Location missing state
3. Invalid email format
4. Invalid phone format
5. Date in future
Scroll through form and fix any red error messages."

"Can't find my report after submitting"
"Try:
1. Wait 10-30 seconds and refresh
2. Search by your pet's name
3. Search by city name
4. Clear all filters
5. Make sure status filter is 'All' or 'Lost'"

"How do I search?"
"Go to petreunion.com/search:
1. Type in search bar (name, breed, color, city)
2. Click 'Filters' to narrow results
3. Select filters (type, location, status, date)
4. Click 'Search' or press Enter
5. Click 'View Details' on any pet card"

"No results found"
"Try:
1. Broader search terms ('dog' instead of 'Golden Retriever')
2. Clear all filters
3. Check spelling
4. Search by location only
5. Make sure status filter is correct"

"Field says 'too long'"
"Character limits:
- Pet Name: 100
- Breed: 100
- Color: 50
- Description: 2000
- Location: 200
Shorten the text to fit."

"What's required vs optional?"
"REQUIRED:
- Pet Type
- Color
- Date Lost
- Location (city AND state)

OPTIONAL (but helpful):
- Pet Name
- Breed
- Size
- Age
- Gender
- Description
- Contact info"
