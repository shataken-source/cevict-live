PetReunion Website - Complete Features Guide

Homepage Features

The homepage (petreunion.com) has 4 main action cards:

1. Search Lost Pets
   - Click to go to search page
   - Search our entire database
   - Filter by multiple criteria
   - View detailed pet information

2. Report Lost Pet
   - Click to report your missing pet
   - Fill out detailed form
   - Submit to database
   - Get matched with found pets

3. Report Found Pet
   - Click to report a pet you found
   - Help reunite with owner
   - Match with lost pet reports

4. Find Shelters
   - Browse local animal shelters
   - Get contact information
   - Find resources in your area

"How It Works" Section:
- Step 1: Report - Submit lost or found pet
- Step 2: Search - Community searches and shares
- Step 3: Reunite - Get matched and reunited

Search Page Features

Search Bar:
- Search by any keyword
- Searches: name, breed, color, city, description
- Real-time search as you type
- Press Enter or click Search button

Filter Options:
- Pet Type: All, Dog, Cat
- Location State: All states or specific state
- Location City: Enter city name
- Status: All, Lost, Found, Reunited
- Date Range: All, Last 24 hours, Last week, Last month, Last 3 months

Filter Button:
- Click to show/hide filter options
- Shows count of active filters
- "Clear Filters" button to reset

Results Display:
- Grid layout (responsive)
- Pet cards showing:
  - Pet name
  - Breed and type
  - Status badge (color-coded)
  - Location with map pin icon
  - Date with calendar icon
  - Description preview
  - "View Details" button

No Results Message:
- Helpful suggestions
- Clear Filters button
- Report Lost Pet button

Report Lost Pet Page Features

Form Sections:

1. Pet Information
   - Pet Name (optional, max 100 chars)
   - Pet Type (required: Dog or Cat)
   - Breed (optional, max 100 chars)
   - Color (required, max 50 chars)
   - Size (optional: Small, Medium, Large)
   - Age (optional, max 50 chars)
   - Gender (optional)
   - Description (optional, max 2000 chars)
     - Character counter shows progress
     - Use for distinctive markings, collar info, microchip

2. When & Where
   - Date Lost (required)
     - Date picker calendar
     - Cannot be future date
     - Cannot be more than 10 years ago
   - Last Seen Location (required, max 200 chars)
     - Format: "City, State" or "City State"
     - Helpful tip shown below field

3. Contact Information
   - Your Name (optional, max 100 chars)
   - Email (optional, max 255 chars)
     - Must be valid email format
   - Phone (optional, max 20 chars)
     - Must be 10-15 digits

Form Features:
- Required fields marked with red asterisk (*)
- Real-time validation
- Character counters on text fields
- Helpful placeholder text
- Error messages appear above form
- Success message after submission
- Auto-redirect to search page after success

Navigation:
- "Back to Home" link at top
- Returns to homepage

Tips for Best Results

When Reporting:
1. Be as detailed as possible
2. Include all required fields
3. Use proper location format
4. Add distinctive markings in description
5. Provide contact information

When Searching:
1. Start broad, then narrow
2. Use filters to refine
3. Try different search terms
4. Check spelling
5. Clear filters if no results

When Contacting:
1. Verify it's your pet
2. Ask for additional photos
3. Meet in public place
4. Trust your instincts
5. Verify ownership details
