PetReunion Facebook Messenger Chatbot - Setup Guide

Overview

This guide provides all the materials needed to configure your PetReunion chatbot for Facebook Messenger. Use these documents to seed your chatbot's knowledge base and configure its responses.

Files Included

1. SYSTEM_PROMPT.txt - Core identity, personality, and behavior guidelines
2. KNOWLEDGE_BASE.txt - Comprehensive information about PetReunion
3. FAQs.txt - Frequently asked questions and answers
4. RESPONSE_GUIDELINES.txt - Tone, style, and response templates
5. EXAMPLE_CONVERSATIONS.txt - Real conversation examples
6. QUICK_RESPONSES.txt - Short answers for common questions

How to Use These Files

Step 1: Upload to Chatbot Platform

For Zapier Chatbots:
1. Go to your Chatbot settings
2. Navigate to "Knowledge Base" or "Documents"
3. Upload each text file as a document
4. The chatbot will learn from these documents

For Other Platforms:
- Copy content into knowledge base sections
- Upload as training documents
- Use as system prompts/instructions

Step 2: Configure System Prompt

Copy the content from SYSTEM_PROMPT.txt into your chatbot's:
- System instructions
- Behavior configuration
- Personality settings

This defines how the chatbot should behave and respond.

Step 3: Seed Knowledge Base

Upload or copy content from:
- KNOWLEDGE_BASE.txt - Core platform information
- FAQs.txt - Common questions and answers
- QUICK_RESPONSES.txt - Quick answer library

Step 4: Configure Response Style

Use RESPONSE_GUIDELINES.txt to set:
- Tone preferences
- Message length limits
- Response formatting rules
- Escalation triggers

Step 5: Train with Examples

Use EXAMPLE_CONVERSATIONS.txt to:
- Show desired conversation flow
- Train on proper responses
- Demonstrate handling of different scenarios

Recommended Configuration

Response Settings

- Max message length: 200 words
- Tone: Warm, empathetic, professional
- Personality: Helpful, encouraging, concise
- Language: English (US)
- Formality: Friendly but professional

Knowledge Base Priority

1. High Priority: FAQs, Quick Responses, Knowledge Base
2. Medium Priority: Response Guidelines, Example Conversations
3. Reference: System Prompt (for behavior configuration)

Escalation Rules

Escalate to human when:
- User is extremely distressed
- Complex technical issues
- Payment/billing questions
- Complaints about platform
- Requests for unavailable information

Testing Checklist

After setup, test these scenarios:

- User reports lost pet
- User reports found pet
- User asks "How do I search?"
- User asks "Is it free?"
- User is emotional/upset
- User needs technical help
- User wants to update report
- User found their pet (success story)

Customization

Brand Voice
- Adjust tone in SYSTEM_PROMPT.txt
- Modify examples in EXAMPLE_CONVERSATIONS.txt
- Update language to match your brand

Platform-Specific
- Add your actual website URL
- Include specific features you offer
- Update coverage areas
- Add partner information

Language
- Translate documents for other languages
- Adjust cultural references
- Localize examples and scenarios

Maintenance

Regular Updates
- Update FAQs as new questions arise
- Add new quick responses based on common queries
- Update knowledge base with new features
- Refresh examples with real conversation patterns

Monitoring
- Review chatbot conversations weekly
- Identify gaps in responses
- Update documents based on user needs
- Refine tone based on user feedback

Integration Tips

Facebook Messenger Specific
- Use quick replies for common actions
- Implement buttons for "Report Lost Pet", "Report Found Pet", "Search"
- Set up persistent menu with main options
- Use typing indicators for longer responses
